# 환경
- Unity 6000.2.9f1

# 진행
- 시작지점, 끝지점이 존재함
- 주어지는 재료로 직접 한정된 숫자의 레일을 만들 수 있음
- 레일은 자유롭게 꼈다뺐다 배치가 가능하나, 열차들이 밟고있는 레일은 뺄 수 없음.
- 끝지점에 도착하면 스테이지 클리어

## 분배
1. 레일 객체에 대한 로직 - 현정
  - 재료 계산
  - 놓을 수 있는 자리 계산(단차가 있다거나...)
  - 레일 깔고, 빼고 배치
  - 기차가 밟고 있는지 여부 (2에게서 받아서 처리)

2. 기차 객체에 대한 로직 - 도원
  - 다음 레일로 이동할 수 있는지 여부 (다음 레일 방향)
  - 이동했을 때 어떻게 꺾여야하는지 계산
